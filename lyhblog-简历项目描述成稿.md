# lyhblog 简历项目描述成稿

更新时间：2026-04-17
适用方向：Python 后端 / Django / 全栈初级岗位
说明：这份文档是给你直接写简历和项目介绍用的成稿。

## 1. 项目名称推荐

不要再只写“个人博客”。

推荐名称：

```text
前后端分离内容管理系统 + AI 写作助手
```

或者：

```text
lyhblog 内容管理系统与 AI 写作辅助平台
```

技术栈副标题可写：

```text
Django REST Framework + Vue3 + MySQL + Claude AI Gateway
```

## 2. 简历上一句话项目介绍

你可以直接写：

```text
基于 Django REST Framework + Vue3 实现前后端分离内容管理系统，支持文章、分类、标签、评论与后台权限管理，并集成 AI 写作助手用于自动生成标题、摘要、标签、提纲和 Markdown 草稿。
```

## 3. 简历项目亮点版（4 条）

这一版最适合直接放简历：

### 版本 A：推荐直接使用

- 基于 Django REST Framework + Vue3 构建前后端分离内容管理系统，完成文章发布、分类标签管理、评论管理、登录注册与后台权限控制等核心功能。
- 封装 Axios 请求层与 Token 鉴权链路，结合 Vue Router 与 Pinia 实现后台访问控制及前后端联调。
- 在后台文章编辑页接入 AI 写作助手，支持根据主题自动生成标题、摘要、标签建议、文章提纲与 Markdown 草稿，并一键回填表单。
- 通过环境变量管理、Nginx + Gunicorn 部署链路与 Linux 环境演练，完善项目从开发到部署的基本闭环。

## 7. 项目介绍长版（可用于 GitHub / 面试 / 自我介绍）

```text
lyhblog 是一个基于 Django REST Framework 与 Vue3 构建的前后端分离内容管理系统。项目支持文章发布与编辑、分类和标签管理、评论管理、登录注册与后台权限控制等基础内容管理能力。在此基础上，我又接入了一个后台 AI 写作助手，能够根据用户输入的主题自动生成文章标题、摘要、标签建议、提纲以及 Markdown 草稿，并在后台文章编辑页中一键回填，提高写作效率。整个项目也覆盖了环境变量管理、前后端联调、Linux 环境部署和 AI 网关异常处理等实践内容。
```

## 8. 你要避免的简历写法

- 后台 AI 写作助手
- 接口联调、权限控制、环境变量、部署演练

重点是：

**说你做成了什么，不要只说你学了什么。**
